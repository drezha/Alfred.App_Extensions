## Aim
This extension will open the currently selected file in Finder in Google Chrome.

## Requirements
- [Alfred app](http://www.alfredapp.com/)
- Alfred Powerpack
- Finder
- Google Chrome

## Usage
1. Download the .alfredextension file
2. Double click to install
